How to run:

Open PostgreSQL in pgAdmin.
Open the Query Tool.
Open or paste finalProj.sql.
Run the script from finalProg.sql to create the tables, insert sample data, create indexes, and run the query library.


This project contains the SQL used to create a relational database for tracking Cybersecurity Club members, events, attendance, and budget transactions. It builds upon my previous proposals and includes sample data, indexes, joins, aggregation, a subquery, and a budget balance query.
