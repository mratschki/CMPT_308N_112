# CMPT-308 Lab 8

Name: Marissa Ratschki

## How to run
1. Open PostgreSQL in pgAdmin.
2. Open the Query Tool.
3. Open lab8.sql.
4. Paste and run the lab8.sql script in sections to create tables, insert sample data, create roles, define stored functions, and create triggers.
5. Run each test and report query separately.

## Description
This lab covers database security and authorization using roles, GRANT, and REVOKE. It also includes a stored function that controls course registration logic and a trigger that automatically records enrollment events in an audit table.
