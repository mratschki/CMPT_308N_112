# CMPT-308 Lab 5

Name: Marissa Ratschki

## How to run
1. Open PostgreSQL in pgAdmin.
2. Open the Query Tool.
3. Run the setup statements from lab5.sql first to create and load the Students, Courses, and Enrollments tables.
4. Run each Part A query one at a time.

## Description
This lab covers subqueries using IN, EXISTS, and NOT EXISTS. It also demonstrates set operations using UNION, INTERSECT, and EXCEPT. Written portions cover functional dependencies, attribute closures, and normalization from 1NF through 3NF.
